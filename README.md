# California Housing Price Prediction

## Project Overview
This project predicts median house values in California using machine learning.

## Workflow
1. Data Loading
2. Data Exploration
3. Missing Value Handling
4. One-Hot Encoding
5. Train-Test Split
6. Linear Regression Training
7. Model Evaluation

## Results
- MAE: 50,701
- R² Score: 0.626
