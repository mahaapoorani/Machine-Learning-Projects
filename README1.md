# California Housing Price Prediction

# Objective
Predict median house values using housing features.

# Dataset
California Housing Dataset

# Features
- longitude
- latitude
- housing_median_age
- total_rooms
- total_bedrooms
- population
- households
- median_income
- ocean_proximity

# Preprocessing
- Filled missing values in total_bedrooms
- One-hot encoded ocean_proximity

# Model
Linear Regression

# Results
- MAE: 50,701
- R² Score: 0.626

# Libraries Used
- Pandas
- NumPy
- Scikit-learn
